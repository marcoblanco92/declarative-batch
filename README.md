# Declarative Batch Container

## Overview

`declarative-batch` è un repository contenitore che raccoglie diversi progetti correlati dedicati alla progettazione e gestione di batch dichiarativi.

L'obiettivo di questo contenitore è raggruppare in un unico punto tutti i repository del progetto, facilitandone la gestione, la navigazione e la collaborazione, pur mantenendo i singoli progetti indipendenti.

## Submodules inclusi

Attualmente, il contenitore gestisce i seguenti submodules:

### spring-declarative-batch
Il progetto principale per la gestione dei batch dichiarativi.

### declarative-batch-yaml-generator
Un plugin che permette di generare automaticamente configurazioni compatibili con i batch, a partire da diagrammi grafici.

### declarative-batch-visualizer
Uno strumento per visualizzare in maniera chiara e immediata la struttura dei batch definiti nei progetti.

> Altri submodules potranno essere aggiunti in futuro man mano che il progetto cresce.
